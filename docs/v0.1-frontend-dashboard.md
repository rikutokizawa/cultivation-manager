# v0.1 Frontend Dashboard

## 目的

Mac ローカルで backend と接続し、最低限の運用導線をブラウザから触れる状態にする。

## 今回実装した範囲

- `frontend/app/page.tsx`
  - ファーストビューに温度、湿度、CO2 濃度、接続状況、タンク水位、栽培画像 2 枚を集約
  - スクロール下に温度、湿度、CO2、水位の時系列グラフ
- `frontend/app/manual-input/page.tsx`
  - 手入力フォーム
  - 最近の手入力一覧
- `frontend/app/export/page.tsx`
  - CSV export 条件指定
  - backend export API への導線
- `frontend/components/`
  - 温度グラフ
  - 状態カード
  - 画像カード
  - 手入力フォーム
  - export フォーム

## 構成メモ

- frontend は Next.js App Router を利用する。
- backend URL は `NEXT_PUBLIC_BACKEND_BASE_URL` で切り替え可能にした。
- 画像表示は backend の `/images` 静的配信を参照する。
- グラフ描画には `recharts` を使用した。
