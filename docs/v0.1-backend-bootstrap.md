# v0.1 Backend Bootstrap

## 目的

Mac ローカル上で FastAPI backend を先に完成させ、後で Raspberry Pi 5 へ移植しやすい土台を作る。

## 今回実装した範囲

- `backend/app/main.py`
  - FastAPI アプリ本体
  - CORS
  - 起動時テーブル作成
  - ダミー画像用の静的配信
- `backend/app/models/`
  - `sensor_records`
  - `image_records`
  - `manual_records`
- `backend/app/api/routes/`
  - `GET /health`
  - `GET, POST /sensor-records`
  - `GET, POST /manual-records`
  - `GET /image-records`
  - `GET /export/sensor-records.csv`
  - `GET /latest-status`
- `backend/scripts/seed_dummy_data.py`
  - 初期テーブル作成
  - ダミー時系列データ投入
  - ダミー画像生成

## Raspberry Pi 移行時の差し替え想定

- `.env` のポートや保存先
- SQLite から別 DB に変える場合は `DATABASE_URL`
- `backend/app/services/seed_data.py` を実センサ収集サービスへ置換
- 画像取得元を USB カメラ処理へ置換

- ダッシュボード画面
